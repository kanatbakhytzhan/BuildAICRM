# Настройка UptimeRobot для Render (без засыпания)

## 1. API

- **URL монитора:** `https://buildcrm-api.onrender.com/health`  
  (не корень `/` — теперь и `/`, и `/health` отдают 200)
- **Интервал:** 5 min
- **Таймаут:** поставьте **60–90 секунд** (на free tier холодный старт до ~50 сек)

## 2. CRM и Admin

- **CRM:** `https://buildcrm-crm.onrender.com`  
- **Admin:** `https://buildcrm-admin.onrender.com`
- **Интервал:** 5 min
- **Таймаут:** обязательно **60–90 секунд**. Иначе запрос попадает в момент холодного старта, сервер ещё не поднялся → 503 → монитор помечает "Down".

В UptimeRobot: редактируйте монитор → найдите поле **Timeout** (или **Response Timeout**) → 90 секунд.

## 3. Почему "не заходится"

На free tier инстансы **засыпают** после ~15 мин без запросов. Первый запрос после сна поднимает сервер **30–60 секунд**. Если открыть сайт и сразу обновлять или если монитор с коротким таймаутом пингует в этот момент — увидите 503 или таймаут. Подождите минуту и обновите страницу.

## 4. Итог

- API: монитор на `/health`, таймаут 90 сек.
- CRM и Admin: таймаут 90 сек.
- В браузере: при первом открытии подождать до минуты, затем обновить.
